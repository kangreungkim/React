### 개발환경 셋팅

1. [Node.js](https://nodejs.org/ko/download/ "Node.js") 에 접속해서
2. 최신버전 최신 LTS 버전: 18.14.0 (includes npm 9.3.1) 설치
3. 설치 후 node 버전 확인

```yaml
$node --version
v18.14.0
```
4. 리액트 프로젝트 생성
```
npx create-react-app blog 
```
